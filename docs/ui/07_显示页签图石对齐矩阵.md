## TS_RS P0 显示页签

### 取景

```text
全显
实时缩放
实时平移
窗口缩放
```

说明：

对应 OCCT Viewer 的 fit all、zoom、pan、window zoom。

### 方位

```text
轴测
俯视
正视
侧视
垂直于
旋转
```

说明：

`垂直于` 表示以当前选中面或参考平面为视图法向。

`旋转` 表示进入 OCCT Viewer 动态视图旋转状态。

P0 必须支持鼠标拖拽旋转视图。这个操作只改变相机 / 视角，不改变 STEP 几何和钢筋数据。

### 渲染

```text
线框
渲染
上色
```

说明：

P0 中 STEP / STP 结构模型仍统一水泥灰。

渲染模式只改变显示方式，不改变业务颜色语义。

`隐线不可见 / 隐线可见` 暂不进入 P0，因为当前业务含义不清。后续如确有需要，再评估 OCCT HLR / hidden line 能力。

### 钢筋显示

```text
激活
钝化
```

说明：

```text
激活：显示钢筋
钝化：隐藏钢筋
```

P0 不把钝化实现为半透明或低饱和显示。

### 切平面

```text
X 向剖视
Y 向剖视
Z 向剖视
设置旋转
```

说明：

P0 先做 X / Y / Z 正交剖视。

`设置旋转` 表示把当前切平面旋转一定角度，不是视图旋转。

OCCT 能力确认：

```text
支持。

X / Y / Z 剖视：Graphic3d_ClipPlane + gp_Pln。
剖切面旋转：旋转当前 gp_Pln 后调用 Graphic3d_ClipPlane::SetEquation。
刷新：V3d_View::Update。
```

P0 先做：

```text
角度输入
预览
确定
取消
```

边界：

```text
这是三维 Viewer 显示剖切，不等于工程图剖切线算法。
```

### 窗口

```text
窗口重叠
窗口水平平铺
窗口垂直平铺
```

说明：

P0 不做。

TS_RS 第一版只做单主窗口 + 单中央 OCCT Viewer，不做多文档 / 多窗口排列。

如后续确实需要多视图，再作为 P2 重新评估。

## 边界

显示页签不负责钢筋生成逻辑。

选择不作为独立顶层页签，而是由 OCCT Viewer + 当前命令 + 状态栏承接。
