# 老图石 Ribbon 外壳对齐规格

## 结论

TS_RS 的 P0 主窗口外壳优先对齐老图石，而不是泛化成普通现代 CAD。

P0 顶层 Ribbon 页签固定为：

```text
开始
显示
钢筋
查询
工程图
```

说明：

```text
查询页签 P0 可以只做空壳 / 置灰 / 占位。
设置不做顶层页签，后续放到开始 / 工程 / 工具菜单或右上角选项。
```

## 外壳与业务的边界

```text
老图石 Ribbon：
  负责用户熟悉的入口和分组。

RebarSmart：
  负责钢筋创建参数、默认值、单位、枚举和算法证据。

OCCT：
  负责 Viewer、显示、选择和几何能力。

Detail：
  负责老图石 CAD 插件兼容导出。
```

不要把“像老图石”误解成：

```text
复刻 MFC/Codejock 运行时。
调用 VisualTS DLL。
按旧 VisualTS 钢筋算法继续逆向。
```

## P0 页签职责

开始：

```text
新建工程
打开工程
保存工程
另存工程
导入 STEP
重新加载 STEP
导出钢筋 STEP
设置入口
```

显示：

```text
全显
实时缩放
实时平移
窗口缩放
轴测 / 俯视 / 正视 / 侧视
线框 / 渲染 / 上色
钢筋激活 / 钝化
X / Y / Z 剖视
设置剖切面旋转
```

钢筋：

```text
这些是第一批外壳按钮。
它们是 UI 入口，不代表 P0A 全部可执行。

参考几何：
  定义平面
  参考线定义
  抽线

创建：
  定距钢筋
  定数钢筋
  角筋
  拉结筋
  插筋
  柱锥环盘面筋
  箍筋
  自定义钢筋

编辑入口：
  P0 不在 Ribbon 展开，主要通过左侧钢筋树右键进入。
```

查询：

```text
P0 保留页签。
按钮可置灰或显示“后续实现”。
后续承载查询、定位、统计、查找相关钢筋等能力。
```

工程图：

```text
新建图纸
选择剖切面
选择投影方向
设置比例
选择钢筋组
生成工程图预览
导出 Detail 包
打开导出目录
```
