# 参考面 / 参考线 UI 设计

本文档记录 TS_RS 中参考面、参考线的 UI 设计。

## 设计结论

参考面 / 参考线创建入口放到钢筋页签。

原因：

```text
参考面 / 参考线主要服务于钢筋布置、延长、截断、投影、定位。
它们不是普通显示功能。
```

## 钢筋页签入口

钢筋页签新增工具组：

```text
参考几何
├── 定义平面
└── 参考线定义
```

说明：

第一版做成轻量小窗口。视觉和字段参考图石，底层几何能力由 OCCT 提供。

当前全局 UI 决策：

```text
钢筋创建命令和钢筋组右键编辑统一使用右侧 Dock。
参考面 / 参考线 / 抽线属于短流程辅助几何工具，仍使用小窗口。
```

不放到右侧 Dock 的原因：

```text
定义平面 / 参考线定义是短流程辅助几何命令。
右侧 Dock 主要服务钢筋生成器长参数面板。
两者混在一起会让当前命令状态变复杂。
```

## 数据与交互

参考几何应作为工程对象保存，而不是只作为临时显示对象。

建议模型：

```text
ReferencePlane
ReferenceLine
ReferenceGeometryId
ReferenceGeometryBinding
```

左侧工程树中可显示：

```text
参考几何
├── 平面
└── 线
```

P0 可先只支持当前命令内使用；P1 再支持持久化、命名、重命名、删除。

## 小窗口交互要求

定义平面窗口：

```text
标题：定义平面
字段：平面类型 / 基准平面 / 距离(米)
按钮：确定 / 取消 / 预览
```

参考线定义窗口：

```text
标题：参考线定义
字段：构造类型 / 结构线 / 结构面 / 距离(米)
按钮：确定 / 取消 / 预览
```

交互规则：

```text
1. 点击字段旁的拾取区域后，主视图进入对应选择模式。
2. 选择对象后，小窗口字段自动填入对象摘要。
3. 点击预览时，在 OCCT Viewer 中显示临时参考几何。
4. 点击确定后，写入工程参考几何对象。
5. 点击取消后，清除临时预览，不修改工程数据。
```

## 技术可行性

OCCT 可表达相关几何：

```text
参考点：gp_Pnt
参考线 / 轴线：gp_Ax1、Geom_Line、TopoDS_Edge
参考面：gp_Ax2、Geom_Plane、TopoDS_Face
显示选择：AIS_Shape 或自定义 AIS_InteractiveObject
```

## 边界

禁止：

```text
把参考面 / 参考线仅当 UI 临时状态，不落工程模型。
在钢筋生成算法里直接依赖 AIS 对象。
把所有图石构造类型一次性塞进 P0。
```

允许：

```text
P0 先实现少数高频构造方式。
用小窗口复刻图石字段。
后续如确有需要，再评估是否增加右侧 Dock 版本。
```
