# 钢筋树右键菜单设计

本文档记录 TS_RS 左侧 Dock 中“钢筋组 / 单根钢筋”右键菜单的设计。

2026-06-16 TODO-016 / P0A 收口说明：

```text
右键菜单是已生成钢筋的编辑入口，不是当前 P0A 的主验收目标。
P0A 只要求一条最小钢筋创建闭环先跑通。
```

## 设计结论

采用方案 A：

```text
钢筋组右键是主要编辑入口。
单根钢筋右键只做定位、高亮、信息、选中所属组。
单根改参 / 删除先不进入 P0。
```

## 为什么这样设计

RebarSmart 的证据主要围绕“钢筋组 / 钢筋几何集 / 编号范围”展开。

它支持对钢筋组做：

```text
全参数编辑
批量更新
修改等级
修改直径
修改间距
修改备注
编号处理
平移复制
对称复制
旋转复制
生成钢筋实体
删除钢筋实体
显隐
查找
信息查看
```

单根钢筋更多用于：

```text
高亮
查看信息
定位
反查所属钢筋组
```

如果第一版直接允许单根删除或单根改参，容易破坏参数化钢筋组的一致性。

## 钢筋组右键与右侧 Dock 切换

右键菜单只负责选择命令，复杂参数不在右键菜单或弹窗里填写。

P0 统一切换到右侧 Dock：

```text
选中钢筋组
  -> 右侧 Dock 显示“钢筋组摘要面板”

右键 -> 编辑全部参数
  -> 右侧 Dock 切到“钢筋组全参数编辑面板”

右键 -> 修改编号 / 修改等级 / 修改直径 / 修改间距 / 修改备注
  -> 右侧 Dock 切到“快速修改面板”
  -> 当前钢筋组自动作为目标对象

右键 -> 平移复制 / 对称复制 / 旋转复制
  -> 右侧 Dock 切到“复制操作面板”
  -> 当前钢筋组自动作为源钢筋组

右键 -> 生成钢筋实体 / 删除钢筋实体
  -> 右侧 Dock 切到“实体操作确认面板”

右键 -> 显示钢筋信息 / 查找相关钢筋 / 查看属性
  -> 右侧 Dock 切到“钢筋组信息 / 查询面板”

右键 -> 定位 / 高亮
  -> 直接执行 OCCT Viewer 动作，不强制切换 Dock

右键 -> 显示 / 隐藏
  -> P0 可直接执行
  -> 如果后续增加范围、轴线/实体、首尾段等复杂选项，再切到显隐面板
```

统一按钮：

```text
预览
应用
取消
```

说明：

```text
1. 右键菜单不承载复杂参数表。
2. 右侧 Dock 是当前钢筋操作面板。
3. 编辑、复制、实体生成/删除都可能需要预览和取消，因此不建议用一次性弹窗。
4. 目标钢筋组必须在 Dock 顶部清晰显示，避免用户误改。
```

## 单根钢筋右侧 Dock

单根钢筋被选中时，右侧 Dock 不能空白。

P0 设计为“单根钢筋信息 / 快捷操作”面板。

显示内容：

```text
所属钢筋组
钢筋编号
直径
等级
间距
长度
保护层
当前显示状态
轴线 / 实体状态
```

快捷操作：

```text
定位
高亮
显示钢筋信息
选中所属钢筋组
显示 / 隐藏
查看属性
```

禁止在 P0 提供：

```text
修改直径
修改等级
修改间距
删除单根
编辑全部参数
```

说明：

单根钢筋右侧 Dock 是“信息面板”，不是“完整编辑面板”。

完整编辑入口必须回到其所属钢筋组，避免破坏钢筋组参数化生成关系。

## 单根钢筋右键与右侧 Dock 切换

P0 规则：

```text
点选单根钢筋
  -> 右侧 Dock 显示“单根钢筋信息面板”

单根右键 -> 显示钢筋信息 / 查看属性
  -> 右侧 Dock 保持或切到“单根钢筋信息面板”

单根右键 -> 选中所属钢筋组
  -> 左侧钢筋树定位到所属钢筋组
  -> 右侧 Dock 切到“钢筋组摘要面板”

单根右键 -> 定位 / 高亮
  -> 直接执行 OCCT Viewer 动作，不强制切换 Dock

单根右键 -> 显示 / 隐藏
  -> P0 可直接执行
```

禁止：

```text
单根右键直接进入修改直径 / 修改等级 / 修改间距。
单根右键直接删除单根钢筋。
```

如果后续确认需要单根局部修改，必须先设计“组内局部 override”模型，再开放对应 Dock 面板。

## 交互边界

右键菜单是 TS_RS 的新 UI 设计，不是 RebarSmart 原生右键菜单复刻。

映射关系：

```text
RebarSmart 工具条命令
  -> TS_RS 钢筋树右键命令
  -> 右侧 Dock 当前操作面板
  -> service / generator
  -> 更新钢筋组参数化对象
  -> OCCT Viewer 刷新预览 / 实体
```

禁止：

```text
UI 层直接写钢筋生成算法
UI 层直接改 OCCT 拓扑对象当业务真相
单根钢筋随意脱离钢筋组参数化模型
```
