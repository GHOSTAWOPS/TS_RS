# 顶层选项卡与功能矩阵

本文档记录 TS_RS 第一版主窗口顶层选项卡和功能分布。

## 总体原则

TS_RS 第一版采用现代 Qt Widgets CAD 布局，但上方模块划分参考图石的操作习惯。

钢筋创建命令参考 RebarSmart。

已生成钢筋的编辑入口优先放在左侧钢筋树右键菜单，不单独设置顶层“编辑”页签。

暂不设置顶层“查询统计”页签。

## 顶层选项卡

第一版顶层选项卡：

```text
开始
显示
钢筋
工程图
设置
```

暂不作为顶层选项卡：

```text
编辑
查询统计
钢筋校审
钢筋导出
钢筋切图
```

说明：

- 编辑：放到左侧钢筋树右键菜单。
- 查询统计：先不做顶层入口，后续可放到底部 Dock 或报表模块。
- 钢筋校审 / 钢筋导出 / 钢筋切图：RebarSmart 有相关模块，但 TS_RS 当前不使用它们作为主线。

## 开始

定位：

```text
工程文件、STEP 模型、保存、导入导出、基础操作。
```

P0：

```text
新建工程
打开工程
保存工程
另存工程
导入 STEP
重新加载 STEP
导出钢筋 STEP
```

P1：

```text
最近工程
工程属性
导出 Excel / CSV
工程打包
撤销 / 重做基础能力
```

边界：

开始页签不放钢筋生成参数。

开始页签不放 `导出 Detail 包`，该入口只放在工程图页签。

撤销 / 重做不作为 P0 按钮承诺。Qt6 技术上支持撤销框架，但 TS_RS 需要先建立统一命令服务和事务边界，第一版只做架构预留。

阶段状态：

```text
已冻结。
后续除非发现必须保留的老图石关键入口，否则不再反复调整。
```

```text
方位：
  轴测

  垂直于
  旋转

  线框
  渲染
  上色

  激活
  钝化

切平面：
  X 向剖视
  Y 向剖视
```

说明：

```text
窗口重叠 / 水平平铺 / 垂直平铺：P0 不做。
```

P1：


边界：

显示页签只管显示和选择，不写钢筋生成算法。

参考面 / 参考线属于布筋辅助几何。显示页签只管理显示 / 隐藏，不提供创建入口。

## 右侧 Dock 与顶层选项卡关系

右侧 Dock 随当前选择或当前命令切换：

```text
未进入命令：显示当前选择对象信息
选择 STEP 面 / 边 / 点：显示几何选择信息
选择钢筋组：显示钢筋组参数摘要和编辑入口
选择单根钢筋：显示单根钢筋信息和快捷操作
执行钢筋创建命令：显示对应 RebarSmart 风格参数面板
执行工程图命令：显示工程图向导步骤
```

## OCCT Viewer 选择交互

已确认方案：

```text
不单独设置“选择”顶层选项卡。
选择点 / 边 / 面 / 实体由 OCCT Viewer 的交互模式实现。
```

交互方式：

```text
1. 用户在当前命令或 Viewer 状态栏中切换选择过滤：点 / 边 / 面 / 实体。
2. 鼠标移动时，Viewer 高亮可选对象。
3. 鼠标点击后，对象进入已选中状态并保持高亮。
4. 右侧 Dock 显示当前选择对象信息。
5. 底部状态栏显示当前选择模式、选择数量、命令提示。
```

说明：

OCCT 技术上支持 sub-shape selection 和高亮。第一版不做独立“选择”顶层页签。

## OCCT Viewer 显示配色

P0：

```text
导入 STEP / STP 模型统一显示为水泥灰。
不管 STEP 文件自身是否带颜色，都不作为主显示颜色。
```

显示语义：

```text
结构模型：水泥灰
钢筋轴线 / 钢筋实体：橙黄
点 / 线 / 面选中：红色
当前高亮 / 强调高亮：洋红
参考几何：青色或黄色半透明
剖切面：半透明蓝灰
```
