# 显示颜色与高亮规则

本文档记录 TS_RS 的 Viewer 基础颜色语义。

## P0 固定颜色

```text
STEP / STP 结构模型：水泥灰
点 / 线 / 面选中：红色
钢筋轴线 / 钢筋实体：橙黄
高亮 / 定位 / 查找结果 / 悬停强调：洋红
参考几何：青色或黄色半透明
剖切面：半透明蓝灰
```

## 语义边界

红色：

```text
用于 OCCT 几何子对象的已选中状态。
例如点、边、面。
不表示错误。
```

橙黄：

```text
用于钢筋对象本体。
包括钢筋轴线和钢筋实体。
```

洋红：

```text
用于强调高亮。
例如鼠标悬停、定位对象、查找结果、右键“高亮”命令。
不表示已选中集合。
```

水泥灰：

```text
用于导入的 STEP / STP 结构模型。
不管文件自身是否带颜色，P0 都统一显示水泥灰。
```

## P1 可扩展

```text
按钢筋直径着色
按钢筋等级着色
按钢筋编号着色
按错误 / 警告状态着色
显示 STEP 原始颜色调试开关
```

P1 颜色模式必须是可切换模式，不能破坏 P0 固定颜色语义。
