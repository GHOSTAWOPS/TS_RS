1. UI 外壳改为老图石 Ribbon：

```text
开始 / 显示 / 钢筋 / 查询 / 工程图
```

2. P0 从 8 个命令缩小为 1 个最小闭环。

3. Detail 包 POC 前置。

4. TopologyBinding Spike 前置。

5. FixDistance / FixNumber P0 命名和边界必须明确为 Centerline P0。

6. 单位 / 枚举 / 默认值文档必须在 UI 面板实现前补齐。

7. UI -> CommandService -> Generator -> RebarModel Transaction 调用链必须单独冻结。

8. IGeometryEngine P1 的投影 / 偏移 / 相交 / 扫掠能力必须规划，不能让 generator 直接 include OCCT。

9. Golden case 和逆向资源复用边界必须补文档，避免“测试绿但真实失败”和商业资源污染。
