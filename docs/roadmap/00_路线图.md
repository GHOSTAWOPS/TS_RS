# TS_RS 路线图

## 当前目标

```text
先把 TS_RS 从 UI 方案推进到可开发架构。
再按 todo.csv 小步进入实现。
```

## 阶段划分

### M0：路线与架构冻结

状态：进行中

目标：

```text
确认 STEP-only + RebarSmart + Detail 路线。
确认 UI 顶层页签、Dock、Viewer 交互。
确认系统模块边界。
确认 todo.csv 任务队列。
```

验收：

```text
docs/ui 完整。
docs/adr 完整。
docs/architecture 完整。
docs/roadmap 完整。
todo.csv 有 next 任务。
```

### M1：RebarSmart 证据与参数模型

状态：进行中

目标：

```text
建立 RebarSmart 资源、PDF、CATNls、INI、IDA 地址索引。
建立 GenerateRebarData 参数结构。
建立 8 个 P0 命令的 Dock 字段证据清单。
```

首批对象：

```text
定距钢筋
定数钢筋
角筋
拉结筋
插筋
柱锥环盘面筋
箍筋
自定义钢筋
```

验收：

```text
每个 P0 生成器有证据入口。
定距 / 定数字段有专项文档。
RebarGrade / RebarDiameter / RebarWeight / GenerateRebar*.ini 字段入口清楚。
```

### M2：纯算法层

目标：

```text
先不接 OCCT，只做可单测的 RebarSmart 纯算法。
```

范围：

```text
SpaceListParser
CalSectionCount
PrioritySpaceDistributor
PriorityListDistributor
GuideCurveZoneCalculator
材料 / 等级 / 直径表加载
```

验收：

```text
单元测试覆盖正常输入、边界输入、异常输入。
算法文档记录证据函数或资源来源。
```

### M3：GeometryEngine + FixDistance / FixNumber

目标：

```text
接入 OCCT 几何能力，但 RebarSmart 生成器只依赖 IGeometryEngine。
```

范围：

```text
IGeometryEngine
OcctGeometryEngine
curveLength
pointAtLength
tangentAtLength
projectWire
offsetWire
assemble
sweepCircle
FixDistanceGenerator
FixNumberGenerator
```

验收：

```text
简单 STEP 上能选面 / 边。
生成定距 / 定数钢筋中心线。
可选生成实体钢筋。
RebarModel 可保存生成结果。
```

### M4：Qt6 / OCCT 操作闭环

目标：

```text
用户能在 UI 中打开 STEP、选择几何、填参数、预览、应用、保存。
```

范围：

```text
主窗口
Viewer
左侧工程树 / 钢筋树
右侧 Dock 框架
FixDistancePanel 或 FixNumberPanel
.tsrebar 保存 / 回读
```

验收：

```text
打开 STEP。
选边 / 面。
生成一组钢筋。
保存工程。
关闭重开恢复。
```

### M5：Detail 包兼容导出

目标：

```text
新系统生成的 RebarModel / DrawingModel 能导出老图石 CAD 插件可读 Detail 包。
```

范围：

```text
DetailPackageReader
DetailPackageWriter
DetailPackageExporter
Detail.xml
DetailNN.stl XML
StbGroup / Std / StbGeo / StbTable / MaterialTable 映射
```

验收：

```text
旧 Detail 包可读。
旧 Detail 包 round-trip 可写。
极简 Detail 包可生成。
人工用 CAD 插件 autoin 验证导入。
```

### M6：扩展筋型与工程图质量

目标：

```text
补齐 RebarSmart 更多生成器和工程图能力。
```

范围：

```text
Polyline
Stirrup
Dowel
Tie
Sweep
Helix
Corner
Frustum
Customize
基础标注
下料表
PDF / SVG / Excel / CSV
```

验收：

```text
真实工程样本逐步建立 golden。
常用生产流程可试用。
```

## 时间口径

```text
3-4 个月：演示闭环 MVP。
6-9 个月：内部试用版。
9-15 个月：生产核心替代。
15-24 个月：高置信度替代。
24+ 个月：极端边界和复杂出图持续补齐。
```
