## 证据优先级

```text
1. 使用说明 PDF / 实际 3DE 界面截图
2. GenerateRebarXXXDlg.CATNls / CATRsc
3. ResourceRebarSmart/*.ini
4. RebarCreation.dll 命令入口
5. AfrRebarObjImp.dll / AfrRebarCore3D.dll / AfrRebarCore.dll 实现类和算法函数
6. RebarSmart 实际运行确认
```

说明：

```text
RebarSmart DLL 只作为逆向证据源。
TS_RS 运行时不得加载或调用 RebarSmart DLL。
```
